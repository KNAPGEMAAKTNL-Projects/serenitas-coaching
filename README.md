# Serenitas Coaching

Website for Marcel Zoeteman — coaching for healthcare professionals.

## Stack

- **Framework:** Astro 6 (Static Site Generation)
- **Styling:** Tailwind CSS 4
- **Deployment:** Cloudflare Pages
- **Icons:** Lucide
- **Analytics:** Cloudflare Web Analytics (cookie-free)

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
